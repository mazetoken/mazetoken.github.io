  • operate in Bitcoin Cash blockchain as peer-to-peer cryptocurrency virtual assets
  • operate without a central issuer and without a centralized promotion or sales developement

    Mining and minting

    Proof-of-work tokens mining is an innovative, decentralized way to distribute tokens
    Mining reward slowly goes to zero for a long period of time

    Download Mminer from Github
    Mminer is continuation of Mistcoin bchd_mist_miner
    Mining tutorial for Windows and Linux is included in the repository
    To mine MAZE or MAZE NFT 0.00001870 BCH / per mining reward is required


    Download smartMaze from Github
    smartMaze miner is based on 0xBitcoin miner
    Mining tutorial for Windows and Linux is included in the repository
    To mine MAZE about 9000 satoshis (smartBCH) / per mining reward is required


    Decentralized minting is another innovative way to distribute SLP tokens

    Download slp-smart-contract code from Github
    Minting tutorial for Windows and Linux is included in the repository
    To mint 1000 REBEL, VANDALS or CARTEL 0.00003400 BCH is required

    Risk: these are not "out of the box" applications, other software and a command line must be used (there is no graphical user interface for now)
